== Description ==

Disable selected plugin (and/or theme) styles and scripts. Set conditions which will be evaluated.

== Installation ==

1. Upload `disable-styles-scripts.zip` to the `/wp-content/plugins/` directory
2. Unzip the archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to `Settings/Disable S&S` and tick checkbox next to the plugin (or theme) style or script that you wish to disable
5. Enter special condition which will be evaluated and if they are true styles or scripts will be disabled
